 # Blinkit Sales Data Analysis (Power BI)

## Project Overview
This project analyzes Blinkit grocery sales data using Power BI to generate business insights and identify trends.

## Key Features
- Interactive dashboard with KPIs: Total Sales, Average Sales, Item Count, Ratings
- Sales analysis by outlet size, location (Tier 1, Tier 2, Tier 3)
- Category-wise performance analysis
- Outlet establishment trend analysis
- Dynamic filters and slicers

## Tools & Technologies
- Power BI
- Power Query
- DAX

## Key Insights
- Identified top-performing product categories
- Analyzed sales variation across outlet types and locations
- Observed trends in outlet establishment and performance

## Files Included
- Blinkit_sales_data.pbix
- Dataset file (Excel)
